AIM :

  If I show my hand in front of the tap it want to flow the water, if I remove my hand it want to stop the water.

COURSE OF ACTION :

In this Experiment we are using IR Sensor, L293D Module, DC Motor, Propeller with Arduino.
IR gives the input of Arduino- if any Object is there ,Arduino read the first condition so the motor will run.
If there is no object the motor will stop.
